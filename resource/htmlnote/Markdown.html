                              • 总论

                              Markdown是一种轻量级的标记语言，用于将文本内容格式化和排版，使其具有一定的结构和样式。markdown文件的后缀名为.md

                              Markdown本身不涉及CSS，是独立于渲染样式的，但Markdown渲染引擎（如网页浏览器、Markdown编辑器等）在将Markdown文本渲染为HTML时，会应用默认的CSS样式来展示文本。这就是为什么在不同的环境中，Markdown渲染的效果可能会有所不同

                              使用 VSCode 编写 markdown 文档

                              在vscode上安装两个插件：Markdown all in one，Markdown Preview Enhanced。前者集成了Markdown的一些基础功能，后者提供了多种css渲染样式，可分页实时预览编辑的效果。

                              Markdown Preview Enhanced的使用：安装完成后，右上角，也就是运行按钮的旁边，将出现一个分页预览的按钮，点击即可分页，其中一页用于编辑md，另一页实时预览

                              右键可激活附带的功能，包括导出为pdf、jpg等各种格式

                              其它问题

                              1. markdown中支持部分HTML语法，但是，使用HTML语法块时，原本的markdown语法将不再生效，该块中只能使用HTML语法
